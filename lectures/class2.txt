Data Types
	- Basic Data (Primitive data types) or raw data
		Numbers
			byte ( 2^7 - 1): byte age, byte marks, byte hourOfDay
			short ( 2^15 - 1): short populationInCity
			int : int numberOfStudentsInUniversity, int age
			long : long age
			float : float bankBalance
			double : double gpa, double weight
			
		Boolean
			boolean : boolean raining, boolean isSick, boolean isClassPresent
		
		Characters
			char ^ * $ : char initial, char symbol, char section
